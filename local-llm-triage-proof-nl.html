De implementatie is een wisselbaar backend: een enkele env-variabele schakelt tussen cloud API, localhost, of een AWS GPU-endpoint.

Met Gemma 4 27B op een AWS g5 instance breidt hetzelfde patroon zich uit naar coach-diagnostiek en golden-validatie — dat dekt de helft van de LLM-aanroepen in de pipeline op self-hosted inference. De code is al geschreven. Alleen de URL verandert.

Nu: abonnement dekt het

Op dit moment draait de pipeline lokaal via Claude Code op een maandelijks abonnement. Cloud LLM-aanroepen vallen binnen het abonnement — nog geen per-token facturering. Dit is de proof-of-concept fase.

reasoning_tokens: 0 en finish_reason: "stop" bevestigen dat het model pure JSON produceert zonder chain-of-thought overhead.

Productie: API-facturering is onvermijdelijk

Een productiepipeline kan niet blijvend draaien op een ontwikkelaarsabonnement. De contractvoorwaarden van Anthropic vereisen API-toegang voor geautomatiseerde productie-workloads. Zodra dit naar de cloud gaat, wordt elke LLM-aanroep een per-token kost.

Dát is het moment dat self-hosted inference zichzelf terugverdient.

Kostenvergelijking per call — productieprijzen

Backend Kosten Latency Toelichting
Cloud API ~$0,02/call ~2s Per-token facturering, vereist voor evolutie/bootstrap/refine
Self-hosted 27B ~$0,32/uur gedeeld ~3-5s Spot GPU, verdeeld over alle triage/coach/critic calls
Lokaal E4B $0,00 ~21s Alleen dev/test — draait op laptop
De overgang is structureel, niet optioneel. Het abonnementsmodel werkt voor proof-of-concept. Productie vereist API-toegang. Elke call site die we naar self-hosted inference verplaatsen is een call site die niet op de productie API-factuur terechtkomt.
07

Veiligheid: validatie-gate

Een kleiner model kan labels hallucineren of ongeldige JSON retourneren. De validatie-gate vangt dit af voordat het de queue bereikt.

1

Parse JSON

Extraheer JSON-object uit het antwoord. Handelt markdown-wrapped output correct af.

2

Valideer enums

Label moet een van 8 bekende waarden zijn. Actie moet een van 4 zijn. Confidence moet een float zijn.

3

Fallback

Bij validatiefout: other / investigate / 0.0. Identiek aan het "geen API-key" pad.

Het slechtste scenario is identiek aan de no-LLM fallback. Een slechte classificatie bereikt de pipeline nooit ongemerkt — het routeert naar menselijke review.

11

Wat 27B op AWS ontsluit

Self-hosted (3 van 6 call sites)

Bewezen

Triage-classificatie

Gestructureerde classificatie in vaste labels. Werkt op 4B. Op 27B: sneller, betrouwbaarder, nul kosten per call.

Ontsloten door 27B

Coach-diagnostiek

Lees pytest-output, produceer diagnostische hints. 27B heeft genoeg redeneervermogen om een stack trace naar een regex-mismatch te herleiden. Slechte hints verspillen refine-iteraties — kwaliteit telt hier.

Ontsloten door 27B

Golden critic

Valideer voorgestelde JSON tegen de contractmatrix. Gestructureerde regelcontrole, geen creatieve generatie. Fout = "opnieuw proberen", geen dataverlies.

Cloud API (3 van 6 call sites)

Blijft in de cloud

Evolutie-runner

Meerstaps graph-redenering met MCP tool-calls. Vereist lange context, tool-gebruik en architecturaal inzicht. De kern-intelligentie van de proof pipeline.

Blijft in de cloud

Bootstrap + Refine

Genereert complete parsercode en chirurgische diffs. Correctheid is binair — een verkeerde regel betekent dat de test nog steeds faalt. Geen graceful fallback mogelijk.

Het principe: self-host voor single-shot gestructureerde taken (classificeren, valideren, diagnosticeren — waar fout = "investigate"). Cloud API voor meerstaps-generatie (waar fout = verspilde compute of kapotte code).
